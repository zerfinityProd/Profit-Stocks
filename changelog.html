3. Technical Refinement Notes

Testimonials Slider Stability (CSS & React)

By mapping a sliding index offset in the React render method (`(reviewIdx + offset) % length`), three cards are mounted simultaneously. Applying fixed widths (`360px`) and heights (`320px`) inside the `.review-card` container ensures that when a user clicks the "Next" or "Previous" buttons, the text swap triggers instantly without altering the container boundary box, preventing layout shifts in the section below.

Semantic CSS Underline Styling

Rather than using fixed inline divider boxes which look detached on fluid viewport sizes, we refactored headings to wrap exactly to content length (`width: fit-content;`). The pseudo-element border is then mapped to `width: 100%`, aligning the blue line directly underneath the final characters of each footer title.
